WHAT THIS MEANS FOR DEVELOPERS
================================================================================

⚠️ CRITICAL: The language dropdown controls BIBLE DATA, not just UI labels.

When user selects Amharic (am):
  ← Loads Bible verses with language_code='am' from database
  ← NOT: Translating English text to Amharic
  ← NOT: Showing English with translated UI labels

This is a data-driven language switch, not a UI-only switch.

================================================================================
KEY PRINCIPLE FOR TEAM
================================================================================

The language selector is NOT just changing UI labels.

It's selecting which LANGUAGE'S BIBLE TEXT to display.

User picks Amharic → sees Amharic Bible verses (not English with Amharic UI)
User picks Arabic → sees Arabic Bible verses (not English with Arabic UI)
User picks Swahili → sees Swahili Bible verses (not English with Swahili UI)

This is data-driven language switching, achieved by querying BibleVerseText
with the language_code filter parameter.
